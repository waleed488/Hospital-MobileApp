# 🏥 MediSync – Hospital Management System

A Flutter-based Hospital Management System built with Firebase that streamlines healthcare management through secure authentication, appointment scheduling, digital prescriptions, and medical record management for patients and doctors.

## Features

- Secure Firebase Authentication
- Role-based access (Patient & Doctor)
- Patient registration and profile management
- Doctor profile management
- Browse and search doctors
- Appointment booking and management
- Digital prescription creation
- Medical records management
- Appointment status tracking
- Biometric authentication (Patients)
- Light & Dark theme support

## Tech Stack

- **Frontend:** Flutter, Dart
- **Backend:** Firebase Authentication, Cloud Firestore
- **State Management:** Provider
- **Local Storage:** SharedPreferences
- **Biometric Authentication:** local_auth

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/yourusername/medisync.git
```

2. Install dependencies

```bash
flutter pub get
```

3. Configure Firebase

```bash
flutterfire configure
```

4. Run the application

```bash
flutter run
```
